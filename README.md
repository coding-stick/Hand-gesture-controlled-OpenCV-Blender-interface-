# Hand gesture-controlled OpenCV Blender interface

This Is Probably the Worst Way to Control 3D… But It Works



### Instructions:

1. Open Blender File (use your own if you have one), name the object "Cube"
2. In the scripting tab, load blenderport.py, open blender console window and run it.
3. Run mainport.py
4. Enjoy!



### Controls:

* one hand - rotation. Open palm- manual rotate. Index and finger only = automatic. Closed = no rotation. 
* 2 hands - squeeze your index and thumbs together to initialize. Stretch 2 hands apart = scale. Move both hands around = translate object.
